## 1. サービス概要

### HERO AIVOチャットボット作成サービスとは

HERO AIVOが提供する**オリジナルチャットボット作成サービス**は、クライアント企業のWebサイト（LP/HP）に**企業オリジナルのAIチャットボット**を統合し、24時間365日の自動接客を実現するサービスです。

### 特徴

#### ホワイトラベル型ソリューション
- **完全カスタマイズ可能**: 会話内容を企業のサービス・商品に合わせて自由に設定
- **ブランドマッチング**: 企業のブランドカラー、ロゴ、トーンに合わせたデザイン
- **自社開発**: 外部サービスへの依存なし、完全自社管理

#### どんなWebサイトにも対応
- WordPress、Shopify、Wix、Squarespace、Webflowなど主要CMS全対応
- 静的HTML、独自CMSにも統合可能
- **たった2行のコードで導入完了**

#### 業種・用途に合わせた会話設計
- BtoB営業サポート
- ECサイトの商品案内
- サービス説明・FAQ対応
- リード獲得・問い合わせ誘導

## 2. チャットボットの機能・仕様

### 主要機能

#### インタラクティブな選択式会話
- **LINE風のUI**: ユーザーが選択肢をクリックして会話を進める直感的なデザイン
- **24時間自動応答**: 人的リソース不要で常時顧客対応
- **スマート会話フロー**: 最大40-50トピックの会話に対応可能
- **リセット機能**: ユーザーが会話を最初からやり直せる

#### ユーザー体験

```
訪問者がLPにアクセス
    ↓
右下にフローティングアイコン表示（企業ロゴ）
    ↓
「オンライン」ステータスで安心感
    ↓
クリックでチャットウィンドウ起動
    ↓
ウェルカムメッセージと選択肢表示
    ↓
選択肢をクリックして会話が進む
    ↓
最終的に問い合わせフォームや詳細ページへ誘導
```

## 4. CMS統合方法

### WordPress統合（詳細ガイド）

#### ステップ1: ファイルのアップロード

**FTP/cPanelを使用**
```
/wp-content/themes/[your-theme]/chatbot/
├── aivo-chatbot.js (26.5KB)
├── aivo-chatbot.css (4.8KB)
└── icon.png
```

**WordPress管理画面から**
- メディア > 新規追加でファイルをアップロード
- アップロード先URLをメモ

#### ステップ2: コードの追加

**方法A: functions.phpを使用（推奨）**

WordPressダッシュボード:
- 外観 > テーマファイルエディター > functions.php

以下を最後に追加:
```php
// HERO AIVOチャットボットの読み込み
function aivo_chatbot_enqueue_scripts() {
    // CSSの読み込み
    wp_enqueue_style(
        'aivo-chatbot-css',
        get_template_directory_uri() . '/chatbot/aivo-chatbot.css',
        array(),
        '1.0.0'
    );

    // JavaScriptの読み込み
    wp_enqueue_script(
        'aivo-chatbot-js',
        get_template_directory_uri() . '/chatbot/aivo-chatbot.js',
        array(),
        '1.0.0',
        true
    );
}
add_action('wp_enqueue_scripts', 'aivo_chatbot_enqueue_scripts');
```

**方法B: ヘッダー/フッターに直接追加**

- 外観 > テーマファイルエディター > footer.php
- `</body>` タグの直前に追加:

```php
<!-- HERO AIVOチャットボット -->
<link rel="stylesheet" href="<?php echo get_template_directory_uri(); ?>/chatbot/aivo-chatbot.css">
<script src="<?php echo get_template_directory_uri(); ?>/chatbot/aivo-chatbot.js"></script>
```

**方法C: プラグインを使用**

- プラグイン「Insert Headers and Footers」をインストール
- 設定 > Insert Headers and Footers
- Footerセクションに以下を追加:

```html
<link rel="stylesheet" href="/wp-content/themes/your-theme/chatbot/aivo-chatbot.css">
<script src="/wp-content/themes/your-theme/chatbot/aivo-chatbot.js"></script>
```

#### ステップ3: 動作確認

- サイトをブラウザで開く
- 右下にチャットボットアイコンが表示されることを確認
- クリックして会話が正常に動作するか確認

### Shopify統合

#### 方法1: テーマエディタを使用

1. **ファイルのアップロード**
   - 設定 > ファイル > ファイルをアップロード
   - `aivo-chatbot.js`、`aivo-chatbot.css`、`icon.png`をアップロード
   - アップロード後のURLをコピー

2. **テーマに追加**
   - オンラインストア > テーマ > アクションメニュー > コードを編集
   - `theme.liquid` を開く
   - `</body>` タグの直前に追加:

```liquid
<!-- HERO AIVOチャットボット -->
<link rel="stylesheet" href="{{ 'aivo-chatbot.css' | asset_url }}">
<script src="{{ 'aivo-chatbot.js' | asset_url }}"></script>
```

#### 方法2: カスタムLiquidセクションを作成

1. セクション > 新しいセクションを追加
2. `aivo-chatbot.liquid` として以下を作成:

```liquid
<link rel="stylesheet" href="{{ 'aivo-chatbot.css' | asset_url }}">
<script src="{{ 'aivo-chatbot.js' | asset_url }}"></script>

{% schema %}
{
  "name": "HERO AIVO Chatbot",
  "settings": []
}
{% endschema %}
```

**最もシンプルな方法**

```html
<!DOCTYPE html>
<html lang="ja">
<head>
    <meta charset="UTF-8">
    <title>Your Site</title>
    <!-- 他のhead要素 -->
    <link rel="stylesheet" href="/chatbot/aivo-chatbot.css">

<body>
    <!-- サイトのコンテンツ -->

    <script src="/chatbot/aivo-chatbot.js"></script>

```

### 統合時の注意点

#### パス指定
- **相対パス**: `/chatbot/aivo-chatbot.js`
- **絶対パス**: `https://yourdomain.com/chatbot/aivo-chatbot.js`

#### ファイルの配置場所
- Webサーバーの公開ディレクトリ内
- CMS/プラットフォームのアセットフォルダ

#### 読み込み順序
1. CSS（ヘッダー）
2. JavaScript（フッター、body閉じタグ直前）

#### キャッシュ対策
- ファイル更新後、ブラウザキャッシュをクリア
- バージョン番号をURLに追加: `aivo-chatbot.js?v=1.1`

## 6. 技術的特徴

### 非エンジニア向け説明

#### 軽量・高速

**ファイルサイズ**
- JavaScript: 26.5KB（Word文書1ページ分程度）
- CSS: 4.8KB
- 合計: 31.3KB

**読み込み速度への影響**
- ほぼゼロ（平均的な画像1枚より軽い）
- ページ速度スコアに影響なし
- SEOにも悪影響なし

#### 既存サイトへの影響ゼロ

**独立した動作**
- 既存のデザインを壊さない
- 他の機能と干渉しない
- エラーが発生してもサイト本体に影響なし

**名前空間の分離**
- すべての要素に専用のID/クラス名
- CSSの競合なし
- JavaScriptの競合なし

#### セキュリティ

**安全な設計**
- 外部サーバーへの通信なし（完全ローカル動作）
- ユーザー情報の収集なし
- Cookie使用なし
- プライバシーリスクゼロ

**XSS（クロスサイトスクリプティング）対策**
- ユーザー入力を受け付けない（選択式のみ）
- HTMLエスケープ処理済み

#### ブラウザ互換性

**対応ブラウザ**
- Chrome（最新版）
- Firefox（最新版）
- Safari（最新版）
- Edge（最新版）
- モバイルブラウザ全対応

**非対応**
- Internet Explorer 11以下（シェア1%未満）
